Step 4 — when does broadcasting work?
# Rule: each dim must be EQUAL or one of them must be 1 a + c → ✓ (4,3)+(3,): 3==3, missing dim treated as 1 d + c → ✓ (4,1)+(3,): 1 stretches to 3, 1 stretches to 4 a + 5 → ✓ scalar applies to every single cell np.ones((2,4)) + np.ones((4,4)) → ValueError! 2≠4, neither is 1
(4,3) + (3,) ✓
1D row matches cols. Repeated down rows.
(4,1) + (3,) ✓
Both stretch. Result is (4,3).
(4,3) + scalar ✓
Scalar applies to every cell.
(2,4) + (4,4) ✗ ValueError
