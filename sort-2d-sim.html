Step 4 — axis rule: which direction does sorting travel?
np.sort(a, axis=0) → ↓ sorts DOWN each column np.sort(a, axis=1) → → sorts ACROSS each row np.sort(a) # default: axis=-1 = axis=1 for 2D
axis=0 → sort ↓ down columns
Each column is sorted top to bottom. Rows are NOT kept together.
axis=1 → sort → across rows
Each row is sorted left to right. Columns are NOT kept together.
Default axis=-1 = axis=1 for 2D
np.sort(a) with no axis = sorts each row. Always specify axis in 2D to be safe.
