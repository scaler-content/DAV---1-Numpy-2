Step 4 — np.sort(a) vs a.sort(): key difference
b = np.sort(a) → returns NEW array, a unchanged ✓ a.sort() → modifies a IN-PLACE, original lost ✗ b = np.sort(a)[::-1] # descending — reverse the result
np.sort(a) — Safe ✓
Returns a NEW sorted array. Original a stays unchanged.
a.sort() — In-place ✗
Modifies a directly. Original data is overwritten!
Descending: np.sort(a)[::-1]
Reverse the sorted result to get descending order.
