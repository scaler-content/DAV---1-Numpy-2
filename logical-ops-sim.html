Logical Operations — np.any() · np.all() · np.where()

Step 1 — build your marks array
marks = np.array([45, 78, 23, 92, 67, 88, 34, 56, 91, 72]) # pass ≥ 35 | distinction > 90
Step 2 — np.any(): did at least ONE student score above 90?
np.any(marks > 90) # True if even ONE element satisfies # ⚡ short-circuits — stops immediately when first match found
 
True ✓ np.any() returns True
np.any(marks > 90) = True
Match found — returns True.
⚡ np.any() short-circuits
Once a match is found NumPy stops — skipped cells stay gray.
Step 3 — np.all(): did EVERY student pass?
np.all(marks >= 35) # True only if ALL elements satisfy # ⚡ short-circuits — stops immediately when first failure found
 
False ✗ np.all() returns False
np.all(marks >= 35) = False
Failure found — returns False.
⚡ np.all() short-circuits
Once a failure is found NumPy stops — skipped cells stay gray.
Step 4 — np.where(): label each student Pass or Fail
np.where(marks >= 35, 'Pass', 'Fail') # NO short-circuit — every element must be labelled → ['Pass','Pass','Fail',...]
 
8 → Pass ✓
students passed
2 → Fail ✗
students failed
np.where(cond, 'Pass', 'Fail')
Labels every element — no short-circuit. Original array unchanged.
np.where() returns a NEW array
marks is never modified. np.where always creates a new labelled array.
